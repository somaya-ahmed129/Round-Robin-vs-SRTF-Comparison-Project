# Round Robin vs SRTF Comparison Project

This project is a CPU Scheduling Simulator that compares:
- Round Robin Scheduling Algorithm
- Shortest Remaining Time First (SRTF)

The application provides:
- Interactive GUI
- Gantt Chart visualization
- Performance metrics comparison
- Fairness and quantum analysis

## Features

- Add and manage processes
- Adjustable Round Robin quantum
- Gantt chart visualization
- Waiting Time, Turnaround Time, Response Time calculations
- Comparison between RR and SRTF
- Fairness analysis
- Modern GUI using Tkinter


## Technologies Used

- Python 3.13
- Tkinter GUI
- Object-Oriented Programming (OOP)


## How to Run

1. Open terminal in project root folder

2. Run the following command:
